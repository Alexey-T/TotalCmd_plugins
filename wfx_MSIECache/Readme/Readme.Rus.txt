Плагин "MSIE Cache Browser" для Total Commander
-----------------------------------------------
Перед Вами плагин для просмотра содержимого кэша Microsoft Internet Explorer
(или браузеров на его движке). В отличие от просмотра Проводником, плагин
отображает кэш в иерархическом виде: список доменов, список сайтов для каждого
домена, список URL для каждого сайта.

- Записи кэша можно просматривать и удалять. Открытие записи открывает
  соответствующий URL в браузере в offline-режиме.
- Чтобы перечитать кэш, откройте псевдо-файл "Перечитать кэш".
- Чтобы копировать URL в буфер, скопируйте запись(и) в "c".


Установка
---------
При наличии TC 6.5+: просто откройте архив, и TC сам установит плагин.
Или же, для установки вручную: распакуйте архив и проделайте установку
как для любого другого FS-плагина: перейдите в меню Настройка -> Плагины ->
Плагины файловой системы -> Добавить, и затем выберите файл IECache.wfx.
