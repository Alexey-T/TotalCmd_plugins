Введение

File Thumbnails - это Lister-плагин для Total Commander, который позволяет просматривать эскизы (миниатюры) различных файлов. Поддерживаются следующие типы файлов:

  • Video File (*.AVI,*.MPG,*.MPEG,*.DIVX,*.OGM,*.MKV,*.M2V,*.DAT,*.VOB,*.ASF,*.WMV)
  • AutoCAD Graphics (*.DWG)
  • CorelDRAW Graphics (*.CDR)
  • Adobe Acrobat PDF (*.PDF)
  • Digital Camera Raw Files (*.CAM,*.CCD,*.CR2,*.CRW,*.RAW)
  • TrueType Fonts (*.TTF)

Важное предупреждение

Полноценная поддержка плагином всех перечисленных типов файлов возможна только при установке дополнительных модулей второго уровня, формат которых совместим с форматом плагинов программы WhereIsIt. Данные модули можно скачать с оффициального сайта http://www.whereisit-soft.com. Однако учтите, что в лицензии на использование следующих плагинов: DescPlugin.dll, DescGraphics.dll и CDDAPlugin.dll сказано буквально следующее:

"Using or distributing these librarires for any purpose other than running them as a part of WhereIsIt is prohibited and in violation of copyright laws".

То есть, использовать эти модули можно только как часть самой программы WhereIsIt.
Поэтому, для соблюдения авторских прав автора WhereIsIt в плагине запрещена загрузка трех вышеуказанных DLL-модулей. Естественно, при этом становится невозможной поддержка многих типов файлов, реализуемая данными модулями, но таково требование автора WhereIsIt, которое я, как автор плагина File Descriptions, вынужден соблюдать.

Установка

  1. Просто откройте архив, и TC сам установит плагин. После появления на экране списка Lister-плагинов переместите плагин в списке на самый верх (присвойте ему высокий приоритет).

  2. Перейдите в меню Конфигурация -> Настройка -> Эскизы и включите опцию При помощи LS-плагинов, установив маску как *.*.

  3. Включите просмотр эскизов: Вид -> Просмотр эскизов

Как подключить плагины WhereIsIt

  1. Во-первых, Вы должны самостоятельно скачать эти плагины с оффициальной страницы. Перейдите на http://www.whereisit-soft.com и скачайте следующие модули:

    • Video Files Thumbnails
    • AutoCAD Plugin
    • CorelDraw Plugin
    • Adobe Acrobat PDF Thumbnails
    • Digital Camera Raw Files
    • TTF Plugin

  2. Распакуйте архивы с модулями в подпапку Plugins главной папки плагина.

  3. Откройте файл Thumbs.ini и убедитесь, что следующие строки в нем не закомментированы: 
        [Plugins]
    1=VideoThumbPlugin.dll
    2=AcadPlugin.dll
    3=CorelPlugin.dll
    4=PDFThumbnailPlugin.dll
    5=DcrawPlugin.dll
    6=TTFPlugIn.dll

Вот и все - плагины должны подключиться при следующем запуске Total Commander.

Примечания

  1. Процесс конфигурирования плагинов WhereIsIt пока не реализован, поэтому плагины будут использовать свои установки по умолчанию.

  2. Модуль DcrawPlugin.dll, обеспечивающий показ миниатюр фотографий с цифровых фотоаппаратов, может не работать в связи с одной особенностью. Дело в том, что модуль использует вспомогательную библиотеку dcraw.dll, путь к которой при загрузке определяет исходя из значения ParamStr(0), т.е. пути к основному .exe-файлу. В связи с тем, что при запуске из-под Total Commander основной .exe-файл - это Totalcmd.exe, данный путь вычисляется некорректно. Поэтому, чтобы модуль загружался, сделайте следующее: скопируйте файл dcraw.dll в каталог Папка_TC\Plugins.
