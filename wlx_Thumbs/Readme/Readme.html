Introduction

File Thumbnails is Total Commander Lister plugin, that allows to display thumbnails for various files.
It is able to handle the following file types:

  • Video File (*.AVI,*.MPG,*.MPEG,*.DIVX,*.OGM,*.MKV,*.M2V,*.DAT,*.VOB,*.ASF,*.WMV)
  • AutoCAD Graphics (*.DWG)
  • CorelDRAW Graphics (*.CDR)
  • Adobe Acrobat PDF (*.PDF)
  • Digital Camera Raw Files (*.CAM,*.CCD,*.CR2,*.CRW,*.RAW)
  • TrueType Fonts (*.TTF)

Important note

For full support of all listed file types 2nd-level plugins are required, which format is compatible with WhereIsIt program plugins format.

You of course may download these WhereIsIt plugins from official site http://www.whereisit-soft.com, but note that in the license of plugins DescPlugin.dll, DescGraphics.dll and CDDAPlugin.dll it's written:

"Using or distributing these librarires for any purpose other than running them as a part of WhereIsIt is prohibited and in violation of copyright laws".

So, due to copyright reasons, File Descriptions plugin prohibites loading of three DLL modules listed above. Of course, now it's impossible to show descriptions for file types supported by these modules, but as the author of File Descriptions plugin, I don't want to violate copyrights of WhereIsIt author.

Installation

  1. Just open archive and TC will install plugin automatically.
    After plugins list opens, put plugin to the top of the list (assign to it high priority).

  2. Go to Configuration -> Options -> Thumbnails
    and check option Get from Lister plugins, set its mask to *.*.

  3. Turn on thumbnails: Show -> Thumbnail view

How to use WhereIsIt plugins

  1. First you need to download them. Go to http://www.whereisit-soft.com and get the following modules:

    • Video Files Thumbnails
    • AutoCAD Plugin
    • CorelDraw Plugin
    • Adobe Acrobat PDF Thumbnails
    • Digital Camera Raw Files
    • TTF Plugin

  2. Unpack modules archives to Plugins subfolder of main plugin folder.

  3. Open Thumbs.ini and make sure the following lines are uncommented: 
        [Plugins]
    1=VideoThumbPlugin.dll
    2=AcadPlugin.dll
    3=CorelPlugin.dll
    4=PDFThumbnailPlugin.dll
    5=DcrawPlugin.dll
    6=TTFPlugIn.dll

That's all - now you need to restart Total Commander.

Notes

  1. Configuring process for WhereIsIt plugins is not yet implemented, so plugins will use they default configuration.
