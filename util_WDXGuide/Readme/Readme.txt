English description
-------------------

This program tests all Content (WDX) plugins installed in Total Commander.
You can easily switch between plugins and see how plugin processes selected file.

Note: "Test filename" can be passed in command line. So it's recommended
to add button to TC toolbar with command Guide.exe and parameters %P%N.


Russian description
-------------------

Программа проверяет работу всех установленных WDX-плагинов к Total
Commander, показывая результат работы каждого плагина для заранее выбранного
файла. Полезна при тестировании плагинов на известном наборе файлов.

Примечание: "Test filename" можно передавать программе в командной строке.
Поэтому рекомендуется создать кнопку на панели кпонок TC с командой Guide.exe
и параметрами %P%N.
