Text Line
Content-плагин для Total Commander
Автор: Алексей Фомин (ledsoft@tut.by)

Описание
--------
Плагин предназначен для отображения одной строки текстового файла.
Есть возможность выбора порядкового номера строки, кодировки текста,
можно заменить часть текста на другой.
Настройки хранятся в файле TextLine.ini
Без файла настроек плагин обрабатывает все файлы ничего не заменяя.
Пример настроек см. в файле TextLine.example.ini
