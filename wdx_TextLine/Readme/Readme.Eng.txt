Text Line
Content-плагин for Total Commander
by: Alexey Fomin (http://ledsoft.narod.ru)

Description
-----------
Plugin is intended to show one line of a text file.
You can select line number, text encoding, you can replace one substring
to another.

Settings are stored in TextLine.ini (TextLine.example.ini is example).
Withour settings file plugin handles all files, and doesn't replace anything.
