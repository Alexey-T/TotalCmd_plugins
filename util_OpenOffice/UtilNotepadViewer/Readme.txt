OpenOffice.org Notepad Viewer
-----------------------------
This program is a converter of OpenOffice.org documents to simple text form
(without any formatting). It also acts as Notepad launcher, so you just run:
"OONotepadView.exe Filename.ODT" and it shows final result in Notepad
(or other program assotiated with text files).

Notes:
- unzip32.dll must be placed into the same folder as OONotepadView.exe
- OpenOffice.org installation is not required at all
- Unicode filenames are not yet supported. Do you need them?

Supported document extensions:
- ODT ODS ODP ODG ODF ODB ODM OTT OTH OTS OTG OTP (OOo 2.x)
- SXW SXC SXG SXI SXD SXM STW STC STD STI (OOo 1.x)
