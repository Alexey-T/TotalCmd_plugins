Introduction

This plugin allows to view OpenOffice.org, MS Word 2007 and some other XML-based documents in Total Commander. Documents are converted to XHTML form, you can see them with most of formatting. Not all document styles may be supported because of underlying XSL stylesheets work.

Supported formats:

  • OpenOffice.org 1.x Writer/Calc/Draw (SXW STW SXC STC SXD STD)
  • OpenOffice.org 2.x Writer (ODT OTT)
  • MS Office Word 2007 (DOCX DOTX)
  • FictionBook (FB2 FB2Z)

In Thumbnails view mode of Total Commander plugin allows to show documents thumbnails (currently only for OOo 2.x and if thumbs are embedded into document).

Warning

Currently plugin cannot work properly in TC Lister mode (most of keys in Lister don't work while plugin is active). But it works OK in TC Quick View mode. Plugin works more correctly in Universal Viewer.

Installation

With Total Commander 6.50 or newer, just open the archive and TC will install the plugin automatically. If you already have plugin installed, you may first need to uninstall the previous version so plugin's detection string will be updated.

DOCX Support

To view DOCX documents you need one of the following:

  • Translator from Clever Age (OpenXML/ODF Translator Command Line Tool). It requires the .NET Framework 2.0.

  • Translator from Novell (OpenOffice OpenXML Translator). It doesn't require .NET, and it works a little faster than the one from Clever Age. Get the odfconverter*.oxt file, it is the ordinary ZIP file.

Unzip the Translator package into any subfolder of the plugin folder. The OdfConverter.exe file should appear. If you didn't unzip the Translator where needed, you'll see the error message box on opening DOCX files. If you have installed the Translator right, you'll see the conversion progress window on opening DOCX.

Configuration

You can edit the following options in the "OOoViewer.ini" file, located in the plugin folder:

Key Meaning
[Options] section
AllowWorkInLister Enables plugin work in TC Lister (F3 mode). (Work in QuickView mode is always enabled.)
Default is 1 (enabled).
ShowImages Enables showing images for FB2 documents. Plugin will start slower.
Default is 1 (enabled).
TempExtension Enables using extension in the temp folder's name. When enabled, folder name for "example.odt" file will look like "C:\Temp\example.odt.tmp", overwise it will look like "C:\Temp\example".
Default is 1.
Note that it must be set to 0 if you use Novell ODF Converter.
