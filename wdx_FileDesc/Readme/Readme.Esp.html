Introducción

File Descriptions es un plugin de contenido para Total Commander, que permite mostrar descripciones de ficheros:
  • Descripciones de Descript.ion/Files.bbs
  • Para ficheros de texto: Contenido del fichero
  • Para ejecutables/DLLs: Información de versión
  • Para ficheros HTML: Contenido de Meta tags y Title
También permite utilizar plugins adicionales, para mostrar descripciones de los siguientes tipos de ficheros:
  • Videos AVI/QuickTime (*.AVI,*.MOV)
  • Videos MPEG (*.MPG,*.MPEG,*.MPE,*.M1V,*.M2V,*.VOB)
  • Imágenes (*.BMP,*.JPG,*.GIF,*.TGA,*.PNG,*.PSD,*.TIFF,*.PCX)
  • Música de flujo (*.MP3,*.MP2,*.MP1,*.OGG,*.WMA,*.WAV,*.VQF,*.AAC,*.APE,*.MPC,*.FLAC,*.CDA)
  • Música de pista (*.MOD,*.XM,*.IT,*.S3M,*.MT2,*.STM,*.MTM,*.UMX)
  • Documentos de MS Office (*.DOC,*.DOT,*.XL?,*.PPT,*.PPS,*.POT,*.OBD,*.WPD,*.SDA,*.FPX,*.MIX)
  • Documentos de OpenOffice.org (*.OD?,*.OT?,*.SX?,*.ST?)
  • Ficheros RPM de Linux (*.RPM)
  • Ficheros Imágenes de Disco (*.PQI)

Importante

Para soportar todos los ficheros indicados se requiere de plugins de 2do nivel. Los plugins de 2do nivel soportados son:
  • Plugins WhereIsIt (*.DLL)
  • Plugins Disk Explorer Pro (*.DFP)
  • Plugins de contenido de Total Commander (*.WDX)

Observación sobre Plugins WhereIsIt. Usted puede descargar estos plugins de la página oficial http://www.whereisit-soft.com, pero observe que la licencia de DescPlugin.dll, DescGraphics.dll y CDDAPlugin.dll expresa:

"Using or distributing these libraries for any purpose other than running them as a part of WhereIsIt is prohibited and in violation of copyright laws".

["El uso o distribución de estas librerías para cualquier propósito distinto al de ser ejecutadas como parte de WhereIsIt está prohibido y es una violación de las leyes de derechos reservados"].

Por tanto, en respeto de los derechos reservados, el plugin File Descriptions prohibe la ejecución de estos tres módulos. Por supuesto, es imposible mostrar descripciones de los tipos de ficheros soportados por estos módulos, pero como autor del plugin File Descriptions, no deseo violar las leyes de derechos reservados del autor de WhereIsIt.

La situación con los plugins de Disk Explorer Pro es mejor, el autor permite el uso libre de sus plugins, excepto el plugin pictview.dfp, pues utiliza librerías comerciales. Por tanto, se incluyen en este paquete los plugins más útiles de Disk Explorer Pro.

Los plugins de contenido para Total Commander se distribuyen libremente, Ud. puede descargarlos desde la página no oficial http://totalcmd.net.

Instalación

  1. Con TC 6.50+, solo abra el archivo y TC automáticamente instalará el plugin.
    Luego que la lista de plugins abra, coloque al plugin al inicio de la lista (le asigna la más alta prioridad).

  2. Vaya a Configuración -> Opciones -> Vistas personalizadas;
    cree una vista nueva llamada "Descripciones" y agréguele una nueva columna;
    presione "+" y seleccione un campo de la lista de FileDesc.

  3. Active la vista personalizada:
    Ver -> Vistas personalizadas -> Descripciones.

Como usar los plugins de WhereIsIt

  1. Primero necesita descargarlos. Vaya a http://www.whereisit-soft.com y descargue los siguiente módulos:

    • Movie Module [Módulo de Videos]
    • Music Module [Módulo de Audio]
    • Linux RPM Description Module [Módulo de descripción de RPM de Linux]
    • Drive Image Description Module [Módulo de descripción de Imágenes de Disco]
    • GEOS File Module [Módulo de ficheros GEOS]
    • GHOST Image Description Module [Módulo de descripción de Imágenes Ghost]

  2. Descomprima los archivos de módulos a la subcarpeta Plugins de la carpeta principal del plugin.

  3. Abra el fichero FileDesc.ini y elimine la marca de comentario a las siguiente líneas: 
      [Plugins]
  10=DescMovies.dll
  11=MusicModule.dll
  12=RPMPlugIn.dll
  13=pqidesc.dll
  14=GeosPlugin.dll
  15=GhostImagePlugin.dll

Eso es todo, ahora debe reiniciar Total Commander.
Para configurar los plugins, utilice el programa Config.exe.

Como utilizar los plugins de contenido de TC

  1. Primero necesita descargarlos. Vaya a la página http://www.totalcmd.net, abra la página "Content Plugins" ("Plugins de Contenido") y descargue los plugins necesarios.

    Algunos plugins útiles son:

    • AudioInfo (incluído en este paquete)
    • AnyTag
    • xPDFSearch
    • Image Info (incluído en este paquete)
    • ImgSize
    • Exif
    • Media
    • ShellDetails
    • OpenOffice.org Info
    • CDR Info
    • CDocProp
    • wlxFont (también trabaja como plugin WDX)

  2. Descomprima los archivos de plugin a la subcarpeta Plugins de la carpeta principal del plugin.

  3. Abra el fichero FileDesc.ini y para cada plugin agregue estas líneas: 
      NN=Path\Plugin.wdx
  NN_detect=*.EXT1,*.EXT2,*.EXT3
  NN_format=["Prefix1"Field1"Suffix1"] ["Prefix2"Field2"Suffix2"]
    Donde: NN es la ID del plugin (un número de 40 a 100), "Path\Plugin.wdx" especifica la ruta del plugin (absoluta o relativa a la carpeta Plugins), NN_detect especifica la lista de extensiones soportadas, NN_format especifica la cadena de formato. La cadena de formato contiene una lista de campos con prefijos y sufijos, prefijo+valor+sufijo y se muestra si el campo no está vacío.

El paso 3 también puede ejecutarse usando el programa Config.exe:
1. Presione el botón Install new y busque el plugin,
2. Seleccione el plugin al final de la lista, presiones el botón Configure y establezca los parámetros del plugin.

Para comprobar la ejcución de los plugins, utilice Config.exe. Si Ud. configuró adecuadamente el fichero FileDesc.ini, el programa mostrará plugins en la lista y le permitirá configurarlos.

Para aplicar una nueva configuración de plugins, Ud. necesita reiniciar Total Commander.
