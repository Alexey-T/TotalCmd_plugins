"UnicodeTest/LockedTest" плагин для Total Commander
---------------------------------------------------
Данный content-плагин показывает несколько полей:

1. "Unicode test"/"Тест юникодности":
  "ANSI"    - файловое имя (но не содержание!) - ANSI-имя;
  "Unicode" - файловое имя (но не содержание!) - Unicode-имя;
  "Ошибка" - файл был не найден.

2. "Locked test"/"Тест чтения":
  показывает, доступен ли файл на чтение, или нет.

3. "Write test"/"Тест записи":
  показывает, доступен ли файл на запись, или нет.


Как установить
--------------
Как и любой контентный плагин - например в диалоге опций TC, вкладке "Плагины". После установки надо настроить новые колонки в диалоге опций, вкладке "Наборы колонок".
