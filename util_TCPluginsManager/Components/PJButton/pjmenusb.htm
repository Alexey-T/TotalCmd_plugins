Description

This unit contains a set of three speed button derived components that either display pop-up menus associated with the buttons or form connected button / menu button pairs. The components are:

  • TPJMenuSpeedButton -- which can display an associated pop-up menu when clicked.
  • TPJLinkedMenuSpeedButton -- links to a master button and displays an associated pop-up menu when clicked.
  • TPJLinkedSpeedButton -- a speed button that can link to and control a slave button that accesses a pop-up menu.

Compatibility

The unit is compatible with Delphi 4. It may work correctly in Delphi 5 but this has not been tested.

Installation

  1. Unzip the component's .zip into a folder of your choice, preserving the directory structure.
  2. Copy the files PJMenuSpeedButtons.pas, PJMenuSpeedButtons.dcr and PJMenuSpeedButtons.res to where you wish to install into Delphi. This will probably be a sub-folder of the $(DELPHI)\lib folder.
  3. Install the unit's components and property editor into the palette by chosing the Component | Install Component menu option to display the Install Component dialog box. Then:
    • Select the Into existing package tab.
    • Click the Browse button next to the Unit file name edit box to display the Unit file name dialog box.
    • Select PJMenuSpeedButtons.pas from the file dialog and click OK to return to the Install Component dialog box.
    • Choose the package into you wish to install from the Package file name drop down list.
    • Click OK.
    • A confirmation dialog box, querying whether it is OK to rebuild and reinstall the package may displayed. Click Yes to proceed.
    • The package will now be built and a dialog box stating that the TPJMenuSpeedButton, TPJLinkedSpeedButton and TPJLinkedMenuSpeedButton components have been installed should be displayed. Click OK to dismiss the dialog box.
  4. You can now delete PJMenuSpeedButtons.pas from the place where you copied it if you wish (but ensure that the .dcu files are not deleted). The component will appear on a palette called DelphiDabbler. The other code in the unit can be made available simply by use-ing the unit in your projects. You can move the components to a different palette as required.
  5. The simplest way to install the help file is to use my Component Help Installer Expert. To install manually, proceed as follows:
    • Copy PJMenuSpeedButtons.hlp to whichever folder you wish to install it to.
    • In Delphi 3 to 6 (where X is the Delphi version 3..6):
      • Edit DelphiX.ohl (in $(DELPHI)\Help), adding the line :Link PJMenuSpeedButtons.hlp to the end of the section labelled "Third party help".
      • Edit DelphiX.ohi (in $(DELPHI)\Help), adding the line :Index DelphiDabbler Menu Speed Buttons=PJMenuSpeedButtons.hlp towards the end of the index section.
      • Edit DelphiX.cnt (in $(DELPHI)\Help), ensuring that the lines :Include delphiX.ohiand :Include delphiX.ohl appear somewhere in the file.
    • In Delphi 7:
      • Edit D7.ohl (in $(DELPHI)\Help), adding the line :Link PJMenuSpeedButtons.hlp to the end of the section labelled "Third party help".
      • Edit D7.ohi (in $(DELPHI)\Help), adding the line :Index DelphiDabbler Menu Speed Buttons=PJMenuSpeedButtons.hlp towards the end of the index section.
      • Edit D7.cnt (in $(DELPHI)\Help), ensuring that the lines :Include d7.ohi and :Include d7.ohl appear somewhere in the file.
    • Using RegEdit open the key HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\Help and add a new string value named PJMenuSpeedButtons.hlp whose value is the path where you copied the help file (probably ($DELPHI)\Help).
  6. The supplied demo programs can be loaded into Delphi and compiled and run as required.
