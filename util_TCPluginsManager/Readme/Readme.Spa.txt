TC PlugMan es un manejador externo de plugins para Total Commander.
Le permite:

 - Ver la lista de todos los plugins Lister/FS/Compresor/Contenido instalados
 - Agregar/remover/configurar plugins (¡más fácil que en TC!)
 - Activar/desactivar plugins (¡TC no puede hacer esto!)
 - Ejecutar/reiniciar TC

Notas
- También le muestra si TC esta actualmente en ejecución; si es así, luego de
  hacer algunos cambios, debe reiniciar TC para que los cambios tengan efecto.
- La característica "Instalar desde archivo" solo opera con WinRAR instalado
  (http://rarlab.com).

Comprobado y funciona en Total Commander V5.5/6.0, p.e. en la versión actual 6.54.
