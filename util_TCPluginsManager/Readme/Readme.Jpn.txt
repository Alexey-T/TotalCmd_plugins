TC Plugins Manager は、TotalCommander(http://ghisler.com), 
専用のプラグイン管理ユーティリィです。 

各種プラグインは http://totalcmd.net. から入手することができます:

主な機能:

 - インストールされたすべてのLS/FS/Packer/Contentプラグインのリストビュー
 - プラグインの追加/移動 (TCよりも容易に出来ます!)
 - プラグインの個別の有効/無効化(TCにはこの機能はありません!)
 - TCの起動/再起動

備考:

1. 設定の変更を行った場合は、変更を反映させる為に、最初に
   TC を再起動する必要があります。

2. "アーカイブからインストール"コマンドの実行には、外部のアンパッカー
　　が必要になります:

   - ZIP からのインストール: WinRAR, UnZip, PKUnZip, WZUnZip, PKZipC;
   - RAR からのインストール: WinRAR, Rar.exe, UnRar.exe.

   推奨されるプログラムは WinRAR (http://rarlab.com)です。WinRARでは、
　 両方のタイプ(ZIP,RAR)を使用することが出来ます。

   また、WinRAR をインストールしなくても、フリーウェアであるアンパッカー
　 UnZip.exe と UnRar.exe がこのプログラムに同梱されています。

3. "アーカイブからインストール" コマンドでアーカイブから直接プラグインを
　　インストールすることが出来ます!　また、ひとつのアーカイブ内の別々の
    フォルダに存在する複数のプラグインをインストールすることも可能です。
　　ただ、複数の.W?X プラグインファイルを同じアーカイブのサブフォルダに
　　入れることは出来ないということに注意してください。

4. "フォルダからインストール" コマンドは、"アーカイブからインストール"の代替
   手段です。パックされたプラグインを入手した場合は"アーカイブからインストール"
   を使用してください。手動でパックを展開した場合は、"フォルダからインストール"
   を使用してインストールしてください。
