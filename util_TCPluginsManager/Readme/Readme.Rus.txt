TC Plugins Manager - это внешний менеджер плагинов к Total Commander.

 Возможности:
 ------------

 - Удобное управление всеми типами плагинов
 - Удобная установка плагинов, в том числе из архивов и вложенных папок
 - Временный запрет загрузки плагинов - то, чего нет в TC!
 - Запуск/перезапуск Total Commander
 - Создание файла-журнала установки плагинов


Замечания:
----------

  1. При работе программа показывает, запущен ли сейчас Total Commander, и если да,
     то после внесения изменений следует перезапустить TC (кнопкой "Перезапуск").

  2. Команда "Установить из архива" требует для работы внешний распаковщик:

     - для ZIP, один из: WinRAR, UnZip, PKUnZip, WZUnZip, PKZipC;
     - для RAR, один из: WinRAR, Rar.exe, UnRar.exe.

     Рекомендуется архиватор WinRAR (http://rarlab.com), как более универсальный.

     Теперь в дистрибутив менеджера включены распаковщики UnZip и UnRar, так что
     нет нужды самостоятельно ставить какие-то программы. Нет необходимости копировать
     эти распаковщики в какие-либо каталоги, они должны запускаться прямо из каталога
     менеджера.

  3. Команда "Установить из архива" позволяет устанавливать большие наборы плагинов,
     из одного или нескольких архивов! Поместите любое число плагинов в отдельные
     подпапки архива (любого уровня вложенности). Примечание: нельзя класть
     несколько .W?X файлов в одну подпапку архива - Вы получите сообщение об ошибке
     в этом случае.

  4. Команда "Установить из папки" есть альтернатива для команды "Установить из
     архива"; можно использовать ее после ручной распаковки набора плагинов в любую
     папку. Но предпочтительнее использовать установку из архива, потому что
     в этом случае автоматически создается правильная струткура папок плагинов.


Параметры командной строки:
---------------------------

  1. /Restart      - Перезапустить Total Commander и завершить работу.
  2. /OrderWLX
     /OrderWDX     - Показать диалог "Порядок плагинов" и завершить работу.
     /OrderWCX            
  3. <ИмяФайла>    - Установить данный плагин.
  4. /Uninstall <ИмяФайла>
                   - Деинсталлировать данный плагин.
