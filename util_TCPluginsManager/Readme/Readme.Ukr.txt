TC Plugins Manager - утиліта для програми Total Commander (http://ghisler.com),
яка дозволяє легко керувати його (TC) плаґінами. Колекцію цих плаґінів ви можете
отримати за адресою: http://totalcmd.net.


Дозволяє:
---------

 - керувати всіма типами плаґінів;
 - інсталювати плаґіни, у тому числі з архівів та вкладених папок;
 - вмикати/вимикати плаґіни - Total Commander це робити не може!
 - запускати/перезапускати Total Commander;
 - протоколювати операції інсталювання та розпакування.


Примітки:
---------

  1. Утиліта показує, чи працює зараз Total Commander, якщо так, то після
     внесення вами якихось змін потрібно перезапустити TC, щоб ці зміни
     набули сили.

  2. Команда "Інсталювати з архіву" потребує для роботи інстальовані зовнішні
     розпакувальники:

     - для ZIP, один з таких: WinRAR, UnZip, PKUnZip, WZUnZip, PKZipC;
     - для RAR, один з таких: WinRAR, Rar.exe, UnRar.exe.

     Рекомендується використовувати WinRAR (http://rarlab.com), який обробляє
     обидва типи.

     До складу дистрибутиву включено вільно поширювані UnZip та UnRar. Копіювати
     їх кудись немає ніякої потреби, вони повинні працювати безпосередньо з папки
     Plugins Manager. Їх можна видалити, якщо ви маєте інстальований WinRAR.

  3. Команда "Інсталювати з архіву" дозволяє встановлювати величезні пакети
     плагінів! Ви можете мати кілька плагінів в одному архіві, в окремих папках
     будь-якої глибини.
     Примітка: Не дозволено мати більше одного файла .W?X в одній папці архіву -
     в цьому випадку ви отримаєте повідомлення про помилку!

  4. Команда "Інсталювати з папки" є альтернативою команди "Інсталювати з архіву".
     Однак, якщо ви маєте архівний пакет плаґіну, рекомендується використовувати
     "Інсталювати з архіву" - у ході інсталяції буде створена належна структура
     папки плаґіну.


Параметри командного рядка:
---------------------------

  1. /Restart               - Перезапустити Total Commander і вийти.
  2. /OrderWLX              - Показати діалог "Порядок плаґінів" і вийти.
     /OrderWDX
     /OrderWCX
  3. <Ім'яФайлу>            - Інсталювати плаґін з даним ім'ям файлу.
  4. /Uninstall <Ім'яФайлу> - Деінсталювати плаґін з даним ім'ям файлу.
