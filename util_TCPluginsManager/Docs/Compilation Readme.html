TC Plugins Manager - Compilation Readme


(15 Apr 2007)

1. Notes

  1. The license for TC Plugman and its sources, except for 3rd party components, is located in License.html file.

  2. 3rd party components are required: Menu Related Speed Buttons. See "Components\PJButton" folder. Homepage of these components: http://www.delphidabbler.com/components.

  3. Compilation was tested in Delphi 5 and 7, was not tested in newer versions. The recommended compiler is D7 because D5 doesn't fully support XP theming for TGroupBox, TPageControl and TOpenDialog/TSaveDialog. Also the FileCtrl.SelectDirectory function has bug in D5.

  4. FS-Plugins Config Tool, to reduce exe size, should be compiled using stripped version of System unit for Delphi 5. Now it's 9 Kb. Get this stripped unit here: http://bonanzas.rinet.ru. TC Plugman itself should be compiled using standard System unit, because it needs exceptions support.

2. Compilation process

  1. Compile and install component: "Components\PJButton\PJMenuSpeedButtons.pas"

  2. Compile "Plugman\FSConfig\Config.dpr" (see Notes section), copy "Plugman\FSConfig\Config.exe" to "Plugman\Config.exe"

  3. Compile "Plugman\Res\Resources.rc". To do it, run:
    brc32 -r Resources.rc

  4. Compile "Plugman\Plugman.dpr". No errors must be during compilation, including missed resources. You may need to correct search paths in Project options, or in dcc32.cfg, so that units from Proc, PJButton folders will be found.

    I recommend to use dcc32.exe: correct search paths in dcc32.cfg and run:

    dcc32.exe -m Plugman.dpr

3. Making distributive

  1. Compile all needed files, see section above. 2 executables must be done: Plugman.exe, Config.exe.

  2. Install required programs:
    • UPX, http://upx.sourceforge.net
      copy upx.exe to %PATH%.
      Note: UPX is not used in latest TC Plugman packages.
    • WinRAR, http://rarlab.com
      install WinRAR so that "start winrar.exe" command may be run from DOS window.

  3. Check "Plugman\Setup\c_setup.bat" that it works correctly. Then run it, it will make "Plugman\Setup\util_TCPlugman.zip".

  4. Test this archive: unpack it to a temporary folder and run Plugman.exe. It must run, autodetect your TC ini paths and allow you to change languages through Options dialog.

  5. Distribution pack must contain files:

    • Plugman.exe - 700+ Kb
    • Config.exe - 9 Kb
    • UnZip.exe
    • UnRAR.exe
    • Language\*.lng - 20+ files
    • Readme\*.txt - 20+ files

    No *.ini files must be in distribution!
